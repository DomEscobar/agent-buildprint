# Phase 04 — Polish + Verify

Run gates:

- typecheck/build if available;
- unit tests;
- smoke/user path;
- screenshot review.

Normal-user screenshot review:

- Can a user identify the next action instantly?
- Is technical detail hidden until after the first result?
- Does the result explain why it matters?
- Is there a clear next action?

Reject if the UI looks like an expert dashboard.
