# Stack And Quality Bar

Default stack:

- Vite + TypeScript.
- React or Svelte.
- CSS modules/plain CSS acceptable.
- Domain logic in `src/domain/`.
- Storage in `src/storage/`.
- Provider/runtime honesty in `src/provider/`.
- UI components in `src/components/`.
- Tests for domain logic and state/export.

If npm install is unavailable, still create TypeScript-flavored modular source and document the blocker. Do not collapse into one huge JS file unless the environment makes framework setup impossible.
