# Product Brief

Microfish helps a user turn messy social-simulation notes into a simple first analysis.

Normal-user framing:

- User pastes notes.
- Product finds people, topics, platforms, and risks.
- Product explains the result in plain language.
- User can download the analysis for later.

Avoid first-screen terms like ontology, provider, deterministic, graph runtime, adapter, OASIS, Zep.
