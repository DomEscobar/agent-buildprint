# First-Time User Journey

User is not an ontology expert.

Journey:

1. User opens the app and asks: “What do I do here?”
2. User pastes messy notes about people and rumors.
3. User clicks one obvious button.
4. User gets a plain-language summary: people, topics, risks, next step.
5. User can expand details if they care.
6. User returns later and sees the saved result.
7. User downloads the analysis.
