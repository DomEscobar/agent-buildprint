# Feature Slice

Implement a local Microfish intake workbench:

- Create/select a project.
- Add seed evidence text.
- Generate a candidate ontology/graph preview.
- Show entities, relationships, assumptions, missing info, and provider status.
- Persist and read back project state locally.
- Export JSON for later simulation prep.

The slice is successful only if a first-time operator can use it end-to-end.
