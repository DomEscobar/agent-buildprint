# User Simulation

Target operator: a researcher preparing a social simulation from messy seed notes.

Journey:

1. I open the local Microfish intake tool.
2. I want to create a project called `Civic Rumor Drill`.
3. I paste seed notes about three actors, two claims, one platform, and a risk.
4. I expect the tool to produce a candidate ontology: actors, claims/topics, relationships, assumptions, and missing information.
5. I want to see whether this came from deterministic local mode or a live provider.
6. I need to correct or regenerate when the notes change.
7. I close/reload and expect my project to still exist.
8. I export JSON for the next simulation stage.
