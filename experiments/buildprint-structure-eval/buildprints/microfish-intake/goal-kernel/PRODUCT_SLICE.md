# Product Slice — Microfish Intake + Ontology

Build a local Microfish slice for a social-simulation research workbench.

A first-time operator should be able to:

1. create or select a project;
2. paste or upload seed evidence text;
3. generate a small ontology/graph candidate from that evidence;
4. inspect entities, relationships, assumptions, and provider status;
5. persist the project locally;
6. reload and see the generated ontology still there;
7. export the ontology/project JSON.

Local-only is fine. No real OpenAI/Zep/OASIS integration is required. If provider controls exist, they must honestly show deterministic/local mode or blocked-live mode.
