# Handover

Write a short honest handover at the end of the run.

Use these headings:

## Current status

Did the local product run? Is the core loop usable? What remains prototype-level?

## Built surfaces

List intake/workbench, simulation setup, runtime/report, interaction/history, persistence, provider boundaries. Mark each built/partial/blocked/not started.

## Verification

Commands run, browser/manual paths checked, screenshots or artifact paths.

## Known defects and blockers

Concrete issues only. Name the user-visible symptom and next repair.

## Next atomic actions

Numbered next steps a human or agent can do without rereading the whole packet.
