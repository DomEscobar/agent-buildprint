# Mission

Create a compact, executable Buildprint for a clean-room portable webapp that maps the Toonflow source product into an implementation plan coding agents can execute without reading the original source.

The target product is a novel-to-storyboard pipeline: chapters -> events -> script plan/script -> assets -> director plan -> storyboard rows -> mock media tasks -> preview manifest.

The package is intentionally narrower than Toonflow. It preserves the core creative pipeline while excluding clone, Electron, live-provider, exact-canvas, and final stitched-video parity.
