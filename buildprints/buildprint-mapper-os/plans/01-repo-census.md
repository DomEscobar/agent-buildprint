# Phase 01 — Repo census

Collect deterministic facts before reasoning:

- languages and frameworks,
- package manager and scripts,
- routes/pages/API handlers,
- data models and migrations,
- integrations and SDKs,
- workers/queues/jobs,
- tests and CI,
- deployment files,
- env var names,
- docs and READMEs.

Output a concise facts summary and preserve unknowns.
