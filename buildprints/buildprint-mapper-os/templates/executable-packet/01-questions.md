# Questions

Ask or answer only questions that change implementation. If unanswered, choose the safest local-first default and record the assumption in handover.

1. Who is the primary user for this build?
2. What is the product's central artifact or main work surface?
3. What is the first usable end-to-end loop?
4. What input/output/state must persist locally?
5. What provider, credential, deployment, destructive, or paid-service boundaries must stay blocked/honest?
6. What would make this embarrassing in a 60-second screen recording?
