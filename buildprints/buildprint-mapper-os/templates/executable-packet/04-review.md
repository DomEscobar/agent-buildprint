# Final Reviewer

Review the product before you explain it.

1. Start fresh and complete the core loop.
2. Reload and verify meaningful state returns.
3. Change inputs and verify output/artifacts change.
4. Click the visible primary controls.
5. Trigger empty, error, blocked-provider, and reset/delete states where possible.
6. Look for generic dashboard smell, fake intelligence, raw JSON, placeholder copy, dead controls, canned output, internal/proof vocabulary, missing persistence, and absent next actions.
7. Fix local, safe, central defects before handover.

Leave only real blockers: credentials, destructive actions, deployment, security review, paid services, environment failure, or large scope.
