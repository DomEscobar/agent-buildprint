# PRE_IMPLEMENTATION_QUESTIONS

Ask at most five blocking questions before coding. Do not ask quality-tier or team-choice questions.

## Blocking Questions

1. Provider mode: live provider or deterministic no-network adapter?
2. Persistence mode: local durable store or named external database?

## Safe Defaults

- Provider mode: deterministic no-network adapter.
- Persistence mode: local durable store.
- Deployment posture: local dev only.
- Destructive actions: disabled.
- Qualification: keep `SELECTED_UNQUALIFIED`.
