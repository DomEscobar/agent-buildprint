# Design Quality Bar

## Product Category

- Operational ingest workflow.

## Taste Direction

- Compact, clear, status-forward.

## Visual Hierarchy

- Input, action, result, and error states are visually distinct.

## Forbidden Generic Patterns

- Generic dashboard cards.
- Raw JSON as the main UI.
- Dead controls.

## Interaction Polish

- Actions must work, show disabled reasons, or record blockers.

## Accessibility Gates

- Keyboard path, labels, focus states, and non-color status are required.

## Responsive Gates

- Desktop and narrow viewport proof required when UI is implemented.
