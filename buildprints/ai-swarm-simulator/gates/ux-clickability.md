# Gate — UX Clickability

Pass only if browser or screenshot evidence proves:

- graph canvas renders real nodes/edges from sample or uploaded input;
- at least one node/edge click opens details;
- upload, sample, build graph, run simulation, generate report controls work or show blocked state;
- loading/blocked/ready states are specific and polished;
- UI does not degrade into static SVG, token bubbles, card list, or generic dashboard.
