# Local RAG Chat Workbench

Start at `BUILDPRINT.md`. This Buildprint is a compact Mapper executable packet for building a local-first RAG chat workbench inspired by Maxkrvo/OllamaChat without depending on the source repository during implementation.

The packet is designed as a faster AI-app regression than the larger swarm simulation Buildprint. It still requires production-grade provider seams, persistence, browser proof, and honest evidence.
