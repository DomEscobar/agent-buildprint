# Minimal Example Expected Shape

A successful coding-agent dry run should create a tiny TypeScript/Node project that can:

1. Load persona canon.
2. Load user memory and self-state from JSON.
3. Create a grounded social draft.
4. Block an ungrounded social draft.
5. Block low-trust sensitive media.
6. Mock-publish only approved drafts.

This folder describes the expected shape; it is not a full implementation.
