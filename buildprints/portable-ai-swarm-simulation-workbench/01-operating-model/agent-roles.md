# Agent Roles

- Product architect: preserve end-to-end workflow, data model, dependency order, and no-fake gates.
- UX/UI craft: produce a serious workbench interface with upload, graph, setup, runtime, report, chat, and history states.
- Integration runtime: own API boundaries, async tasks, subprocess/worker controls, polling, provider adapters, and failure behavior.
- Data persistence: own projects, uploads, extracted text, graphs, simulations, configs, logs, reports, history, export, delete, and restart/readback proof.
- Security boundary: review uploads, secrets, provider logs, destructive actions, subprocess controls, and user-data retention.
- Test and verification: own acceptance matrix, negative tests, browser proof, provider proof labels, and ledger entries.
