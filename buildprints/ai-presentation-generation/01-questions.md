# Questions

Blank answers are allowed. They authorize AI best-fit decisions unless the choice is credentialed, irreversible, expensive, destructive, or product-defining.

## Product posture

- Trusted local only, private authenticated, or public webapp?
- Should this prioritize solo desktop use, team API service use, or both?
- Is compatibility with Presenton-like API semantics required or only the user workflow?

## Provider choices

- Which text providers should be live first: OpenAI-compatible, Ollama, Gemini, Anthropic, Azure, Bedrock, Vertex, or others?
- Which image sources should be live first: DALL-E/OpenAI-compatible, Gemini, Pexels, Pixabay, local generation, or upload-only?
- Should ChatGPT-account sign-in be treated as in-scope, blocked, or replaced by BYOK provider configuration?

## Deck and export choices

- Required export formats: PPTX, PDF, images, HTML, or all?
- Should export rely on LibreOffice/Chromium, a JS PPTX renderer, or another runtime?
- Must imported PowerPoint templates become editable templates in the first pass?

## Security and deployment

- What data may be stored locally?
- Are uploaded documents allowed to leave the machine through providers?
- What auth/tenancy is required before non-local exposure?
