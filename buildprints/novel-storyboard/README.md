# Novel Storyboard Buildprint

Start at `BUILDPRINT.md`. This packet routes an agent through a proof-gated rebuild of a novel-to-storyboard production workbench.

The target is not a generic graph demo. The delivered product must feel like a visually strong storyboard tool with ordered shot frames, selected-frame inspection, media/review states, durable episode data, authenticated runtime and honest provider boundaries.

Use `blueprint.yaml` as the machine-readable contract, follow the active phase in `03-phases/phase-index.yaml`, and record proof in `.buildprint/evidence/evidence-ledger.jsonl`.
