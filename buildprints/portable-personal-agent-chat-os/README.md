# Portable Personal Agent Chat OS

Mapper OS executable Buildprint packet for a self-hosted personal agent chatbot OS.

Start from `BUILDPRINT.md`. It routes setup through `01-questions.md`, `02-project-setup.md`, `blueprint.yaml`, `03-phases/phase-index.yaml`, and `03-phases/phase-flow.md`.

## Scope

The packet preserves the original product scope: streaming chat, provider routing, tools, skills, MCP adapter, memory/compaction, subagent delegation, token telemetry, WebUI/API workbench, durable local persistence, and safety/claim boundaries.

## Validation status

Packaged evidence is seed-only. Runtime implementation evidence belongs in `.buildprint/evidence/evidence-ledger.jsonl` after phase-flow artifacts exist.
